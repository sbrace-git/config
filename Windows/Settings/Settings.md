# Windows Settings

### 文件夹选项

此电脑 > 查看更多(`...`) > 选项

- 常规
    - 打开文件资源管理器时打开 `此电脑`
    - 隐私
        - [ ] 显示最近使用的文件
        - [ ] 显示常用文件夹
        - [ ] 显示来自 Office.com 的文件  
        - 清除文件资源管理器历史记录 `清除`
- 查看
    - [x] 显示隐藏的文件、文件夹或驱动器
    - [ ] 隐藏已知文件类型的扩展名
### 设置

- 辅助功能 > 键盘 > 粘滞键
    - [ ] 粘滞键的键盘快捷方式
